# Connectors

```{list-table}
:header-rows: 0
:widths: 20 30

* - {doc}`Apache Airflow connector <airflow>`
  - Run Airflow tasks as Union tasks.
* - {doc}`AWS Sagemaker inference connector <sagemaker>`
  - Deploy models and create and trigger inference endpoints.
* - {doc}`File sensor connector <file-sensor>`
  - Detect files appearing in your local or remote filesystem.
* - {doc}`Google BigQuery connector <bigquery>`
  - Execute queries on Google BigQuery.
* - {doc}`MemVerge Memory Machine Cloud (MMC) connector <mmc>`
  - Execute Union tasks using MemVerge Memory Machine Cloud.
* - {doc}`OpenAI ChatGPT connector <chatgpt>`
  - Send prompts to ChatGPT.
* - {doc}`Snowflake connector <snowflake>`
  - Query data in Snowflake.
* - {doc}`Spark on Databricks connector <spark>`
  - Submit Spark jobs to the Databricks platform.
```
